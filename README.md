# plash-plash
Packaging plash with plash

```
$ plash --from-github ihucos/plash-plash
macbook:/home/ihucos/plash# plash init
macbook:/home/ihucos/plash# plash -A
plash: fetching 100%
plash: WARNING: not verifying image signature, gpgv not in PATH
plash: extracting...
plash: ignoring dev file: ./dev/null (Operation not permitted)
macbook:/home/ihucos/plash#
macbook:/home/ihucos/plash# 
```
